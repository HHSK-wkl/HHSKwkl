# De quarto brand-extensie voor HHSK

Installeert de brand-extensie van HHSK of geeft instructies voor het
installeren van de templates

## Usage

``` r
get_brand_hhsk()

get_template_hhsk()
```

## Value

Installeert de brand extensie in de working directory

## Examples

``` r
if (FALSE) { # \dontrun{
 get_brand_hhsk()
 get_template_hhsk()
} # }
```

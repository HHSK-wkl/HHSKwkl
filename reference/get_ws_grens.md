# Waterschapsgrens

Waterschapsgrens als bestand

## Usage

``` r
get_ws_grens(pad = "data")
```

## Arguments

- pad:

  een string voor het pad op de ws_grens.gpkg op te slaan. ordt
  aangemaakt als het nog niet bestaat. Default is de map *data*.

## Value

`get_ws_grens()` slaat ws_grens.gpkg op de gekozen locatie.

## Examples

``` r
if (FALSE) { # \dontrun{
get_ws_grens("data")
} # }
```

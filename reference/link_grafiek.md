# HTML link naar grafiek van een meetpunt

HTML link naar grafiek van een meetpunt

## Usage

``` r
link_grafiek(mp, link_tekst = NULL)
```

## Arguments

- mp:

  De meetpuntcode

- link_tekst:

  Optionele tekst

## Value

HTML-tekst met een link naar een grafiek

## Examples

``` r
if (FALSE) { # \dontrun{

link_grafiek("S_0067", "eigen_tekst")

} # }
```

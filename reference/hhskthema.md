# HHSK thema

Dit is een HHSK-thema voor ggplot

## Usage

``` r
hhskthema()

hhskthema_kaart()

hhskthema_oud()

hhskthema_kaart_oud()
```

## Value

Een thema voor ggplot

## Fonts

Voor het correct weergeven van fonts in Rstudio of documenten moeten
extra stappen worden genomen. Zie
<https://systemfonts.r-lib.org/articles/systemfonts.html> voor een
toelichting.

# Kleuren HHSK

De kleuren van de huisstijl van HHSK. Gebruik de toevoeging \_d, \_m of
\_l voor donkere, midden of lichte variant. De originele lichtere
kleurvarianten zijn niet altijd geschikt voor visualisaties. Deze zijn
opgenomen met de toevoeging \_orig.

## Usage

``` r
kleuren_hhsk

blauw_d

blauw

blauw_m

blauw_l

oranje

oranje_m

oranje_l

grijs_d

grijs

grijs_m

grijs_l

groen_d

groen

groen_m

groen_l

wit

tekstgrijs

blauw_m_orig

blauw_l_orig

groen_m_orig

groen_l_orig

oranje_orig

oranje_m_orig
```

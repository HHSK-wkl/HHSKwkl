# HTML-link naar meetpuntpagina met meetgegevens

HTML-link naar meetpuntpagina met meetgegevens

## Usage

``` r
link_meetgegevens(mp, link_tekst = NULL)
```

## Arguments

- mp:

  De meetpuntcode

- link_tekst:

  Optionele tekst

## Value

HTML-tekst met een link naar een grafiek

## Examples

``` r
if (FALSE) { # \dontrun{
link_meetgegevens("S_0067")
link_meetgegevens("S_0067", "eigen_tekst")

} # }
```

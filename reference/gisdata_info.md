# Informatie over Open GIS data van HHSK

Bevat een overzicht van een groot deel van de open GIS data van HHSK met
de bijbehorende URL's om de online data op te halen.

## Usage

``` r
gisdata_info
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 60
rows and 7 columns.
